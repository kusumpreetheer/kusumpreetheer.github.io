Source Code

This source code represents the source code for The Chair Posture Corrector. We started by setting up the hardware for our chair posture corrector, which included an ultrasonic sensor to measure the distance to my back and a buzzer or vibration motor for alerts, all connected to an Arduino board. We then handled the wiring, ensuring the sensor's power and signal pins were correctly connected to the Arduino, alongside the connections for the buzzer or motor. For the programming part, we initialized the pin settings for the sensor and the output device. In our main loop, the ultrasonic sensor continuously measures distance, and we compare this to a predefined threshold for good posture. If the distance exceeds this threshold, indicating poor posture, our Arduino code triggers the buzzer or motor, alerting us to correct our posture. This project was a blend of hardware setup and software programming, aimed at tackling the common issue of maintaining good posture.
